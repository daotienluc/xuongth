Kết nối đến PostgeSQL từ PHP, Laravel, NodeJS

1. Kết nối đến PostgreSQL từ PHP

- Cấu hình extention trong php

Mở php.ini rồi bỏ dấu ; ở trước 2 thư viện pdo_pgsql và pgsql. Mục đích là để 2 thư viện này được nạp cùng với php để bạn dùng nó kết nối qua server POSTgreSQL.

- Stop và start lại Apache để 2 thư viện được nạp vào

- Trong folder htdocs (là folder gốc của webserver  Apache), hãy tạo folder postgresql-php (hoặc tên gì cũng được)

- Trong folder mới vừa tạo, tạo mới 1 file vd1.php và code: 

  • < ?php
  • $connect_str = “host=localhost port=5432 dbname=aa user=postgres password=123 options=’–client_encoding=UTF8′”;
  • $db = pg_connect($connect_str);
  • ?>
  • < ?php
  • $sql =“SELECT * FROM nha_sx”;
  • $data = pg_query($db, $sql);
  • if (!$data) { echo “An error occurred.\n”;exit;}
  • ?>
  • < ?php while ($row=pg_fetch_assoc($data)) {?>
  • < p> < ?php echo  $row[‘ten’]?> < /p>
  • < ?php } ?>

- Tạo project laravel

  • Cài composer
  • Chạy lệnh tạo project
  • Composer create-project laravel/laravel postgre_laravel
  • Chạy project
  • Chạy lệnh trong folder project: php artisan serve

- Cấu hình kết nối db

- Mở file .env và khai báo kết nối

  • DB_CONNECTION=pgsql
  • DB_HOST=127.0.0.1
  • DB_PORT=5432
  • DB_DATABASE=aa
  • DB_USERNAME=postgres
  • DB_PASSWORD=123

- Tạo bảng sessions để Laravel dùng

- Định nghĩa route lấy dữ liệu trong routes/web.php

  • Route::get(‘/nhasx’, function () {
  • $query = \DB::table(“nha_sx”);
  • $kq = $query->get();
  • return view(‘nhasx’, [‘nhasx’=> $kq]);
  • });
  • Tạo view
  • < h1>Danh sách nhà sản xuất< /h>
  • @foreach($nhasx as $nsx) 

  • {{ $nsx->ten}}

  • - Tạo file index.js: tạo file index.js và code

  • const app = express();
  • app.set(“view engine”, “ejs”);
  • app.set(“views”, “./views”)
